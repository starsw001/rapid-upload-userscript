秒传脚本安装教程

2021-11-09

此文档提供 Win/安卓 平台下的完整安装流程, 适用于无法正常安装和使用脚本的用户

*IOS 用户可尝试使用 秒传网页版工具 (百度秒传)

*相关的脚本主页: 百度版秒传脚本, 阿里版秒传脚本

*相关的网盘页面: 百度网盘, 阿里云盘

*教程仅为参考, 实际安装方案并不唯一, Win 端包括火狐, Edge 等多种浏览器, 以及暴力猴插件, 均可正常运行脚本, 安卓端目前仅推荐 Kiwi 浏览器

21.9.7 更新: 添加阿里版秒传脚本主页

21.7.31 更新: 网盘资源更新: Kiwi v93.0.4577.18, Tampermonkey v4.13

Windows 端 (电脑端)

安装 Chrome 浏览器

前往 Chrome 官网 下载安装最新版 Chrome 浏览器

安装 Tampermonkey 油猴插件 (Chrome)

  • 方法 1: 在线安装/一键安装
  1. 访问 谷歌商店 安装插件, 若无法访问请使用方法 2
  • 方法 2: 本地安装
  1. 下载提供的油猴插件 zip 压缩包(版本 4.13): 蓝奏云
  2. 解压 Tampermonkey v4.13.zip, 得到 Tampermonkey v4.13 文件夹, 内含一些零散的文件
  3. 打开 chrome 浏览器, 在地址栏输入 chrome://extensions/ , 回车
  4. 勾选页面右上角的 "开发者模式" 选项, 再点击左上角出现的 "加载已解压的扩展程序" 按钮
  5. 选择第 2 步中解压出来的 Tampermonkey v4.13 文件夹, 然后在 chrome://extensions/ 中看到下图即可(注意打开右下角的按钮启用插件)

安卓端 (手机端)

安装 Kiwi 浏览器

  • 方法 1: 前往 酷安/谷歌 play 等应用商店安装 (尽量安装最新版本)
  • 方法 2: 下载安装提供的 apk 文件 蓝奏云

安装 Tampermonkey 油猴插件 (Kiwi)

  • 方法 1: 在线安装/一键安装
  1. 访问 谷歌商店 安装插件, 若无法访问请使用方法 2
  • 方法 2: 本地安装
  1. 下载提供的油猴插件 crx 安装包(版本 4.13): 蓝奏云
  2. 打开 Kiwi 浏览器, 点击右上角的选项按钮 (图示 1) , 可以看到 "扩展程序" 选项和 "桌面版网站" 选项 (图示 2) , 点击"扩展程序" 选项
  3. 打开右上角的 "开发者模式", 并点击 "Load(+)" 按钮, 选择第一步下载的 crx 文件 ( 图示 3 ), 再打开扩展页内的开关启用插件 ( 图示 4 ), 若弹出权限提示点确定即可 ( 图示 5 )

  • 使用 Kiwi 打开 对应的网盘页, 并勾选上一节中的 "桌面版网站" 选项, 即可看到秒传按钮:
